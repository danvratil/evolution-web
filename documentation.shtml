Reporting Bugs

If you are sure that you have found a bug in Evolution, please take the time to submit a report at http://bugzilla.gnome.org/. Try to fill in as much detail as you can regarding the circumstances that lead to the problem and provide version information.

Kindly check if your bug has been reported before by using the search functionality of Bugzilla.
